# Patryk Stępień — Portfolio

Personal portfolio site built with **Blazor WebAssembly (.NET 10)**, hosted on GitHub Pages.

🌐 **Live:** https://patrykstepienpro.github.io/

## Tech

- Blazor WebAssembly + .NET 10
- Bootstrap 5 + custom CSS (dark theme)
- i18n — PL/EN language switching
- GitHub Actions — automatic deploy to GitHub Pages

## Run locally

```bash
dotnet watch
```
